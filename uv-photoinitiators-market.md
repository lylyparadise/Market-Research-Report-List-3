<p><ul><li>Free-radical Type Photoinitiator</li><li>Cationic Type Photoinitiator</li></ul></p>
<p><p>UV Photoinitiators are chemical compounds that facilitate the curing process of UV-curable materials by initiating the polymerization reaction when exposed to UV light. There are two main types of UV Photoinitiators: Free-radical Type Photoinitiators and Cationic Type Photoinitiators. Free-radical Type Photoinitiators create free radicals when exposed to UV light, which initiate the polymerization reaction in materials like coatings and inks. On the other hand, Cationic Type Photoinitiators generate cations that drive the curing process in materials like adhesives and composites.</p></p>
